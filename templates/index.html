About This App

This weather app is built using Python Flask as part of a learning project. It allows users to check real-time weather updates for any city by integrating with a weather API to fetch and display accurate data. This project was developed by Areeb Mehdi (15728) under the guidance of our instructor, Ms. Aemon, at Iqra University.
